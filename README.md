## Project Setup
Clone the project from GitHub repository :

      git clone https://github.com/fajarutamaa/backend-challenge-4.git
      
change directory :

      cd backend-challenge-4

Install all package dependencies :

      npm install

Compile and hot-reload for development :

      npm run dev

## Tech Stack
- [x] Express Js
- [x] Prisma
- [x] PostgresSQL 16
